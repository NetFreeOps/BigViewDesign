# BigViewDesign
a tool for bigview by drag

> [ ] 简单登录功能
> [ ] 将dataV集成到bigview中  
> [ ] 将echart集成进去  
> [ ] 将treejs集成进去  
> [ ] 集成signalR  
> [ ] 实现拖拽功能  
> [ ] 实现图层功能  
> [ ] 导出功能  
## 1. 实现拖拽功能
> 1. 鼠标按下，记录鼠标位置
> 2. 鼠标移动，计算偏移量，更新数据
> 3. 鼠标松开，停止记录鼠标位置
## 2. 实现图层功能
> 1. 每个图层可以调整透明度
> 2. 每个图层都能独立显示隐藏
## 3.导出功能
> 1. 直接导出为VUE文件
> 2. 导出图片，作为效果图用
